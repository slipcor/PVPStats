# Permission Nodes

The following nodes can be used:

Node |  Permission | Default
------------- | ------------- | -------------
pvpstats.cleanup | remove duplicate names | 游리 op
pvpstats.config | read and alter config values | 游리 op
pvpstats.count | have your kills counted | 游릭 true
pvpstats.debug | start and stop debugging | 游리 op
pvpstats.debugkill | manually add a kill | 游리 op
pvpstats.migrate | migrate from one database type to another | 游리 op
pvpstats.opmessages | receive OP warnings when something is going wrong | 游리 op
pvpstats.purge | purge older entries | 游리 op
pvpstats.reload | reload the plugin | 游리 op
pvpstats.top | get other player's statistics | 游릭 true
pvpstats.nonewbie | the player does not need noob protection | 游릭 true
pvpstats.wipe | wipe statistics completely | 游리 op
