# Permission Nodes

The following nodes can be used:

Node |  Definition | Default
------------- | ------------- | -------------
pvpstats.cleanup | allows to remove duplicates | 游리 op
pvpstats.config | allows to read and alter config values | 游리 op
pvpstats.count | have your kills counted | 游릭 true
pvpstats.debug | allows to debug | 游리 op
pvpstats.debugkill | allows to manually add a kill | 游리 op
pvpstats.migrate | allows to migrate from one database type to another | 游리 op
pvpstats.opmessages | receive OP warnings when something is going wrong | 游리 op
pvpstats.purge | allows to purge older entries | 游리 op
pvpstats.reload | allows to reload the plugin | 游리 op
pvpstats.top | get other player's statistics | 游릭 true
pvpstats.nonewbie | the player does not need noob protection | 游릭 true
pvpstats.wipe | allows to wipe statistics completely | 游리 op
